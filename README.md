# Homebrew Tap for Canvas CLI

This is the official Homebrew tap for [Canvas CLI](https://github.com/jjuanrivvera/canvas-cli), a powerful command-line interface for Canvas LMS.

## Installation

```bash
# Add the tap
brew tap jjuanrivvera/canvas-cli

# Install Canvas CLI
brew install canvas-cli

# Verify installation
canvas version
```

## Upgrading

```bash
brew upgrade canvas-cli
```

## Uninstalling

```bash
brew uninstall canvas-cli
brew untap jjuanrivvera/canvas-cli
```
